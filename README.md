# jubilant-pancake-numbers

A numberdle web application based on [math.tools API](https://math.tools/api/numbers/).

## Features

- **Numberdle**: Find the day of the day through a fun mini-game
- **Search Bar**: Search for facts about any number you're curious about
- **I'm Feeling Lucky**: Get facts about a random number

## How to Use

1. Open the [app's website](https://paul8710.github.io/jubilant-pancake-numbers/)
2. You can play numberdle by entering any number
3. Use the search bar to enter any number and discover its facts
4. Click "I'm Feeling Lucky!" to get random number facts

## Technologies Used

- html, js and css for the website
- math.tools API for number facts
- Responsive design that works on desktop and mobile devices
